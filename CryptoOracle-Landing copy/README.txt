# 🚀 CryptoOracle Dashboard

CryptoOracle is a cross-platform desktop dashboard built with AI-powered predictive analytics, sentiment scoring, and financial insight. It helps crypto traders, analysts, and enthusiasts stay ahead of market moves with elegant visualizations and actionable recommendations.

## ⚙️ Features

- 📈 Machine Learning-based price forecasting
- 💬 Sentiment analysis from news and social data
- 🧠 Buy/Hold/Sell recommendation engine
- 🧪 Leaderboard ranking crypto assets by performance and sentiment
- 🖥️ PyQt5 desktop interface with charts and dropdowns
- 📦 Cross-platform executables for Windows, macOS, and Linux
